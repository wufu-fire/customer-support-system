# 宠物用品售后网站 MVP 站点地图

## 1. 信息架构总览

```text
/
|- 首页 Home
|  |- 品牌与服务介绍
|  |- 核心入口（商品展示 / 售后申请 / 工单查询）
|
|- 商品展示 Products
|  |- 商品列表 /products
|  |  |- 分类筛选
|  |  |- 关键词搜索
|  |- 商品详情 /products/:id
|     |- 商品图片
|     |- 规格参数
|     |- 使用说明
|     |- 售后政策摘要
|     |- 去售后申请按钮
|
|- 售后支持 Support
|  |- 售后申请 /support/request
|  |- 申请成功 /support/request/success
|  |- 工单查询 /support/track
|  |- 工单结果 /support/track/result
|
|- 法务与说明 Legal
|  |- 隐私政策 /legal/privacy
|  |- 服务条款 /legal/terms
|
|- 后台 Admin（内部）
   |- 登录 /admin/login
   |- 工单列表 /admin/tickets
   |- 工单详情 /admin/tickets/:id
```

## 2. 全局导航规则

- 顶部导航：`首页`、`商品`、`售后申请`、`工单查询`
- 右上角固定语言切换：`EN | 中文`
- 页面底部：隐私政策、服务条款、联系邮箱
- 移动端：导航折叠菜单 + 语言切换保持可见

## 3. 页面优先级（MVP）

P0（必须上线）：
- 首页
- 商品列表
- 商品详情
- 售后申请
- 工单查询
- 申请成功页
- 查询结果页
- 后台工单列表
- 后台工单详情

P1（可并行或次上线）：
- 隐私政策
- 服务条款
- 统一错误页（404/500）

## 4. URL 与路由建议

- 公共前台使用 `/`、`/products/*`、`/support/*`、`/legal/*`
- 内部后台统一放在 `/admin/*`
- 避免中英文双 URL，语言通过 i18n 文案切换实现
